# Car Rental Frontend

Simple Angular frontend for a car rental system (Admin + Customer).

## Version
- App: `0.0.0`
- Angular CLI: `21.2.8`
- Angular: `21.2.x`
- PrimeNG: `21.1.6`

## Quick Run
1. Install dependencies:
```bash
npm install
```
2. Start dev server:
```bash
npm start
```
3. Open:
`http://localhost:4200`

## Test Accounts
- Admin: `admin@admin.com` / `password`
- Customer: `customer@customer.com` / `password`

## Useful Commands
- Build:
```bash
npm run build
```
- Tests:
```bash
npm test
```
